# Zomato Multi-Restaurant Cart - Product Requirements Document

This folder contains the comprehensive PRD for Zomato's Multi-Restaurant Cart feature, designed to enable users to order from 2-3 nearby restaurants in a single cart with optimized delivery logistics.

## Contents

- **Zomato_Multi_Restaurant_Cart_PRD.md** — Full PRD document including problem statement, user stories, functional specs, logistics engine, edge cases, and KPIs

## Key Features

- Multi-restaurant cart with 2km distance constraint
- Optimized rider batching and multi-stop delivery
- Multi-taxation engine with per-restaurant GST handling
- Comprehensive edge case management
- Detailed KPIs for adoption, AOV, delivery time, and financial metrics
